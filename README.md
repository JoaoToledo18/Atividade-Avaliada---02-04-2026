# EMPRESA DE COBRANÇA

- O título para cobrança é recebido e, em seguida, é inserido no ‘sistema de cobrança’.
- O sistema emite a 1a carta de cobrança e paralelamente disponibiliza as informações
de contato do devedor para os profissionais do callcenter tentarem fazer contato por
telefone. Os procedimentos ‘1a carta cobrança’ e ‘cobrança telefone’ são simultâneos.
- Se a 1a carta cobrança obtiver retorno é iniciada a negociação, caso contrário é
enviada a 2a carta cobrança.
- Se a cobrança por telefone conseguir fazer o contato é dado início à negociação. Se
houver interesse a negociação do pagamento é feita.
- Se a cobrança por telefone não tiver sucesso e a 2a carta cobrança não tiver retorno
um cobrador externo é acionado.
- Se o cobrador externo localizar o devedor e este tiver interesse em negociar a
negociação do pagamento é iniciada. Caso contrário o título é protestado.
- Se houver um acordo e o seu respectivo pagamento, os valores são transferidos para
a empresa contratante e o processo de cobrança é finalizado. Caso contrário, o título é
protestado.
- Se, após o protesto, o título for pago, os valores são transferidos para a empresa
contratante e o processo finalizado. Caso contrário, o título é devolvido e o processo
finalizado.
